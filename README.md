# These are tools I'm developing to help work on SCDIF data
Many years ago I was given a very small sample SCDIF dataset. I'm finally getting around to using that dataset along with the publicly available documentation to develop these tools. Since the dataset is very small and not something I can distribute, I've developed a script that will generate arbitrary SCDIF data and a parallel tool to validate that data. The sample dataset did contain some inconsistencies when compared with the specification. I've tried to documnent those inconsistencies in the AI skill. The synthetic data does should be to standard.

## Folder: [SCDIF Documentation](scdif_documentation)
This folder contains a couple of tools:

[**J-10+Senate+CSS+Data+Interchange+Format+(SCDIF)+v5.pdf**](scdif_documentation/J-10%2BSenate%2BCSS%2BData%2BInterchange%2BFormat%2B(SCDIF)%2Bv5.pdf) is a publicly available description of the SCDIF. You can view the original at this [SAM.gov link](https://sam.gov/workspace/contract/opp/f0421f2e6b984e6da373556b3ef562f0/view).

[**scdif-explorer.html**](scdif_documentation/scdif-explorer.html) is a tool to explor the data structures in the SCDIF spec. You will need to download the file and open it in a browser.

This is a skill developed with Claude. I provide it in two formats - a skill folder and a single skill markdown file. I have only done cursory testing on this with other AI systems.
[**scdif**](scdif_documentation/scdif/) this the folder version of the skill.
[**scdif-skill-bundle.md**](scdif_documentation/scdif-skill-bundle.md) and [**scdif-skill-bundle.html**](scdif_documentation/scdif-skill-bundle.html) a markdown version of the skill, and an html version for easier human consumption.
