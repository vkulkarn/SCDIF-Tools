# These are tools I'm developing to help work on SCDIF data


## Folder: SCDIF Documentation
This folder contains a couple of tools:

**J-10+Senate+CSS+Data+Interchange+Format+(SCDIF)+v5.pdf** is a publicly available description of the SCDIF. You can view the original at this [SAM.gov link](https://sam.gov/workspace/contract/opp/f0421f2e6b984e6da373556b3ef562f0/view).

**scdif-explorer.html** is a tool to explor the data structures in the SCDIF spec.

### AI Skills
This is a skill developed with Claude. I provide it in two formats - a skill folder and a single skill markdown file. I have only done cursory testing on this with other AI systems.
**scdif** this the folder version of the skill.
**scdif-skill-bundle.md** and **scdif-skill-bundle.html** a markdown version of the skill, and an html version for easier human consumption.
